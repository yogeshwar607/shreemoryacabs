## Converting SVG to Other Formats

### Option 1: Using Online Tools (Easiest)
1. Go to https://cloudconvert.com/svg-to-png
2. Upload the SVG file
3. Set dimensions (e.g., 1080x1080 for Instagram)
4. Download PNG

### Option 2: Using Design Tools
1. Open SVG in **Canva** (free)
2. Resize to required dimensions
3. Export as PNG or JPG

### Option 3: Using Command Line (Mac)
```bash
# Install ImageMagick
brew install imagemagick

# Convert to Instagram profile (500x500)
convert -background none logo-instagram-profile.svg -resize 500x500 logo-instagram.png

# Convert to Instagram post (1080x1080)
convert -background none logo-social.svg -resize 1080x1080 logo-instagram-post.png
```
